### What is this repository for? ###
This is for teaching different protocols on Zealand Business Academy (zealand.dk)

Prototypes are for:
* UDP
* TCP
* MQTT
* HTTP

### How do I get set up? ###
Installation
* pip install pytest
* pip install amqtt
* pip install asyncio
* pip install paho-mqtt
* pip install pytest-timeout

Run in terminal:
* pytest -v -s 

Run in VSCodium:
* [shift]+[ctrl]+b and select "pytest run all"
